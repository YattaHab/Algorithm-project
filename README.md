# Algorithm-project
solving a problem using both naive and optimized solations

## The "Budget Shopping" Problem
You have $ dollars. Find if you can buy exactly two items from two different stores 
(arrays) that cost exactly S total.

Formal:
Given two arrays A (size n) and B (size m), find a ∈ A and b ∈ B such that a + b = S.
